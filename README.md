# Workflow Builder UI

A custom workflow builder built using React without any external diagram libraries.

## Features
- Start, Action, Branch, End nodes
- Drag & drop nodes
- True / False branching
- Add, delete, edit nodes
- SVG curved connections
- Save workflow as JSON

## Tech Stack
- React (Vite)
- JavaScript
- CSS

## How to Run

```bash
npm install
npm run dev
